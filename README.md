# EDRenum-BOF

Identify common EDR processes, directories, and services. Simple BOF of [Invoke-EDRChecker](https://github.com/PwnDexter/Invoke-EDRChecker).

## Compile 

BOF
```
x86_64-w64-mingw32-gcc -c EDRenum-BOF.c -o EDRenum-BOF.o
```
